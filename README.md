## How install
This APP required someone dependencies they are:
1. mongodb
2. nodejs
3. bower
3. npm
4. git

When you have ready the dependencies just follow this tasks.

You should type this:
1. `git clone https://github.com/G33N/SCYS.git`
2. `cd SCYS`
3. `npm install`
4. `bower install`

End.
